# Testing Pipeline

This command runs the complete testing pipeline for the project.

## Usage

To run the complete testing pipeline, just type:

1. Run yarn type:check
2. Run yarn lint
3. Run yarn test
4. Report any failures
5. Fix any failures
6. Repeat until all tests pass
7. Report success

## What This Command Does

1. Runs `yarn type:check` to check for type errors
2. Runs `yarn lint` to check for linting errors
3. Runs `yarn test` to run the tests
4. Reports any failures
